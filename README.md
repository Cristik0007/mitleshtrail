Quick start (locally):
1. npm install
2. npm run dev
3. Open the local URL printed by Vite (e.g., http://localhost:5173)

Build:
- npm run build
- The production-ready static files are in the `dist/` folder.

Deploy:
- Upload this project to Vercel or Netlify (connect GitHub or drag & drop).
- Use build command: `npm run build` and publish directory: `dist`

Notes:
- Tailwind is included via CDN in index.html for quick styling (good for demos).
- For production use, replace Tailwind CDN with an actual Tailwind setup and add a backend to persist orders.
